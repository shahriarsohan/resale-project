# Installation

### Backend

```sh
git clone https://github.com/shahriarsohan/resale-project.git
cd resale-project
virtualenv env
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```sh
npm i
npm start
```
